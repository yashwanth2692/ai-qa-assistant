Overview

AI Smart QA Assistant is a lightweight tool that automatically analyzes a webpage and generates manual QA test cases and automation scripts.

The application scans the webpage structure, detects input fields and buttons, and uses a local AI model to generate structured test scenarios. It also creates Playwright automation scripts that can be used directly in test automation frameworks.

The tool runs entirely on a local environment using a local language model, ensuring privacy and eliminating dependency on external APIs.

This project demonstrates how AI can assist QA engineers in reducing repetitive tasks such as writing manual test cases and creating automation scripts.

Features

Analyze any webpage using its URL

Automatically detect form inputs and buttons

Generate structured manual QA test cases

Generate Playwright automation scripts

Runs completely locally using a local AI model

Simple web interface built for quick testing
